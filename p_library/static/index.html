Задания из Практики модуля D4

Сервер развёрнут на платформе Heroku. Генерит случайные заполняющие фразы по заготовке.

GET https://c5python.herokuapp.com/api/generate/ возвращает JSON с ключом "message" и значением в виде одного сгенерированного сообщения.

GET https://c5python.herokuapp.com/api/generate/<num:int> возвращает JSON с ключом "message", который содержит список из num сгенерированных сообщений по той же схеме.
